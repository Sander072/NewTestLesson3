# Туториал по языку разметки MarkDown

## Как добавить зачёркивание
**Чтобы добавить зачёркивание в разметку MarkDown, используйте следующую конструкцию:**

```
две тильды `~` до и после текста
```

Пример:

~~Хе-хе~~
